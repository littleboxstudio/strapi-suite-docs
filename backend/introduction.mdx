Visit the [What is Littlebox Strapi Suite?](/before-starting) section to learn more about the plugin’s purpose.
Below is the list of features that the plugin adds to Strapi. It’s ideal for building and managing institutional websites and blogs.

- [Menus](/backend/guides/menus): powerful menu management tool to create, customize, and organize menus for seamless navigation.
- [Translations](/backend/guides/translations): simplify and organize string translations for efficient multilingual content management and smooth integration.
- [Slug](/backend/guides/slug): optimize URLs with smart, SEO-friendly slugs for better rankings and cleaner links effortlessly.
- [Sitemap](/backend/guides/sitemap): a way to set the properties to generate the sitemap.
- [Page attributes](/backend/guides/page-attributes): a way to set page properties like templates or parent pages to automatically create full slugs.
- [Parameters](/backend/guides/parameters): effortlessly manage and customize settings with a intuitive and flexible solution.
